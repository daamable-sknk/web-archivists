# 디자인 명세

## 디자인 컨셉

**하이브리드 미니멀**

레퍼런스에서 선택한 요소:
- **neworder.xyz** → 텍스트 중심 레이아웃, 카드형 구성
- **birdcall.online** → 이미지 메뉴, 외부링크 화살표(↬), 업데이트 날짜

핵심 원칙:
- 텍스트가 주인공
- 이미지는 보조적으로 (메뉴/섹션 구분용)
- 외부 링크는 `↬` 화살표로 명확히 표시
- 마지막 업데이트 날짜 표시

## 컬러 팔레트

**모노크롬 (흑백 + 회색)**

```
배경색:     #ffffff (흰색)
텍스트:     #000000 (검정)
링크:       #000000 (검정, 밑줄로 구분)
포인트:     #f0f0f0 (밝은 회색 배경)
보더:       #e0e0e0 (회색 선)
호버:       #666666 (중간 회색)
```

- 컬러 없이 흑백만 사용
- 회색 배경으로 섹션 구분 또는 강조
- 링크는 밑줄로 구분 (색상 변화 없음)

## 타이포그래피

### 폰트 선택
- **한글:** Pretendard (가독성 좋고 무료)
- **영문/코드:** IBM Plex Mono (기술적인 느낌)

### 폰트 크기
```
h1 (제목):     2.5rem (40px)
h2 (섹션):     1.5rem (24px)
h3 (서브):     1.25rem (20px)
본문:          1rem (16px)
작은 텍스트:   0.875rem (14px)
메타 정보:     0.75rem (12px)
```

### 행간
- 본문: 1.7
- 제목: 1.3

## UI 요소

### 링크 스타일
- 기본: 검정색 + 밑줄
- 외부 링크: `텍스트 ↬` 형태로 표시
- 호버: 회색으로 변경 (#000 → #666)

### 버튼
```css
/* 기본 버튼 */
border: 2px solid #000;
padding: 1rem 2rem;
background: transparent;
font-weight: 500;

/* 호버 */
background: #000;
color: #fff;
```

### 구분선
```css
border-top: 1px solid #e0e0e0;
margin: 2rem 0;
```
