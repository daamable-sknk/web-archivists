# 디자인 명세

## 디자인 컨셉

**미니멀 대시보드**

핵심 원칙:
- 텍스트가 주인공
- 컴팩트한 원페이지 레이아웃 (스크롤 최소화)
- 외부 링크는 `↬` 화살표로 명확히 표시
- 좌우 분할 구조 (사이드바 + 메인 콘텐츠)

**모노크롬 (흑백 + 회색)**

## UI 요소

### 링크 스타일
- 기본: 검정색 + 밑줄
- 외부 링크: `텍스트 ↬` 형태로 표시
- 호버: 회색으로 변경 (#000 → #666)

### 버튼 (신청하기)
```css
/* 신청 전 상태 (회색) */
border: 2px solid #ccc;
background: #f5f5f5;
color: #999;

/* 신청 가능 상태 (검정) */
border: 2px solid #000;
background: transparent;
color: #000;

/* 호버 (반전) */
background: #000;
color: #fff;

/* 신청 마감 상태 (검정 배경) */
background: #000;
color: #fff;
```

### 커리큘럼 카드
- 2x2 그리드 레이아웃
- 호버 시: 그림자 효과 + 살짝 위로 이동
- 패딩: 2rem
- 보더: 1px solid #e0e0e0

### 구분선
```css
border-top: 1px solid #e0e0e0;
```

## 반응형

### 브레이크포인트
- 모바일: ~640px
- 태블릿: 641px ~ 1024px
- 데스크탑: 1025px~

### 1024px 이하 조정
- 사이드바가 상단으로 이동 (세로 배치)
- 사이드바 width: 100%
- 메인 콘텐츠도 width: 100%

### 640px 이하 조정
- 타이틀 크기 축소: 1.75rem → 1.5rem
- 커리큘럼 카드: 2x2 그리드 → 1열 세로 스택
- 좌우 패딩: 1.5rem
