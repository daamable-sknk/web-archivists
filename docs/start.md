# 아키비스트는 웹을 꿈꾸는가

```
아카이브의 미래를 설계하는 이들을 위한 웹 리터러시 실천 교육

"오늘날 아키비스트에게 웹은 어떤 존재일까?"
"웹은 아카이브일 수 있을까?"

『아키비스트는 웹을 꿈꾸는가?』는
기록학을 전공했지만 웹에는 아직 익숙하지 않은 실무자들을 위한 재교육 프로그램입니다.

아카이브과 웹의 경계, 저장과 연결, 그리고 협업의 의미를 되묻고,
작은 웹사이트를 직접 만들어보며, 디지털 아카이브의 가능성과 한계를 체험합니다.
```

## 개요

- 목표
    - 기록학을 전공했지만, 웹이 여전히 낯선 아키비스트를 위한 재교육
    - 아카이브에 관심이 많고, 이를 웹으로 표현하고 싶어하는 이들
    - 웹이 가진 아카이브적 속성, 아카이브가 웹을 통해 완성될 수 있음을 이해하기
    - 실천적 구성 : 단순한 형태의 웹사이트 하나를 만들어보기
- 준비물
    - 개인 노트북
    - 여러가지 계정 : Google, Github, Neocities, [Are.na](http://are.na/)
    - 코드 편집기 : Sublime Text
- 다루게 될 기술/도구
    - 웹 코드 기초 : HTML, CSS, JS
    - 웹 문서 : Markdown
    - 정보 공유 및 정리 : Google Drive, Presentation, Spreadsheet
    - 협업 : Github
    - 웹 빌더 플랫폼 : Neocities, [Are.na](http://are.na/)
    - (필요시) Google Gemini, ChatGPT, Claude 등 나에게 적합한 AI
