# 아키비스트는 웹을 꿈꾸는가

> "오늘날 아키비스트에게 웹은 어떤 존재일까?"  
> "웹은 아카이브일 수 있을까?"

기록학을 전공했지만 웹에는 아직 익숙하지 않은 실무자들을 위한 재교육 프로그램입니다.

## 프로젝트 개요

아카이브와 웹의 경계, 저장과 연결, 그리고 협업의 의미를 되묻고, 작은 웹사이트를 직접 만들어보며, 디지털 아카이브의 가능성과 한계를 체험하는 4주 과정의 교육 프로그램 웹사이트입니다.

## 프로젝트 구조

```
web-archivists/
├── index.html          # 메인 페이지
├── curriculum/
│   └── index.html      # 커리큘럼 페이지
├── css/
│   └── style.css       # 스타일시트
├── assets/
│   ├── favicon.ico     # 파비콘
│   └── og-image.png    # Open Graph 이미지
└── docs/               # 프로젝트 문서
    ├── project-plan.md # 웹사이트 개발 계획
    ├── design-spec.md  # 디자인 명세 및 스타일 가이드
    ├── content.md      # 웹사이트 콘텐츠
    └── progress.md     # 개발 진행 내역 및 버전 히스토리
```
